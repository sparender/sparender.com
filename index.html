SPA Render

Sparender is an open-source tool, that helps modern websites beeing indexed properly by search engine such as: Google, Bing, Yahoo, Yandex...

Sparender addresses common SEO (Search Engine Optimization) issues introduced with new javascript frameworks (aka SPAs / Single Page Applications). Sparender was tested with the following libraries / frameworks:

  • AngularJS, ReactJS, Web Components (including Google Polymer), VueJS, EmberJS, etc...

How does it work?

Sparender works because it understands and executes the Javascript complexity of your SPA and renders only plain HTML for bots. In this way, robots will easily crawl and index your page content. This process is done on the fly and only requires a minimal configuration on the server side.

Test it

Note that you are testing your website against a test server, which has some limitations and long delays. You may need to wait around 15 seconds to get a response for a new page. Configuration

The following code shows how to configure Apache to redirect some bots to sparender for karibou.ch website. Note that _escaped_fragment_ will be automatically added by some bots (such as Google bot) if your index.html contains the meta tag: <meta name="fragment" content="!"> inside the head tag. Similar configurations can be found for other HTTP servers. 

<IfModule mod_rewrite.c>
RewriteEngine On

  <IfModule mod_proxy_http.c>
    RewriteCond %{HTTP_USER_AGENT} googlebot|yandexbot|baidupsider|bot|crawler|spider|pingdom|monitis|teoma|facebook|bing|yahoo|embedly|quora|outbrain|pinterest|vkshare|validator [NC,OR]
    RewriteCond %{QUERY_STRING} _escaped_fragment_
                              
    # Proxy HTTP requests to sparender, only if it's a request for HTML
    RewriteRule ^(?!.*?(\.js|\.css|\.xml|\.less|\.png|\.jpg|\.jpeg|\.gif|\.pdf|\.doc|\.txt|\.ico|\.rss|\.zip|\.mp3|\.rar|\.exe|\.wmv|\.doc|\.avi|\.ppt|\.mpg|\.mpeg|\.tif|\.wav|\.mov|\.psd|\.ai|\.xls|\.mp4|\.m4a|\.swf|\.dat|\.dmg|\.iso|\.flv|\.m4v|\.torrent|\.ttf|\.woff))(.*) http://test.sparender.com/http://karibou.ch%{REQUEST_URI} [P]
  </IfModule>

</IfModule>
